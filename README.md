个人自用网页导航

请将图片文件保存在images文件夹下，js文件与css文件保存在与index.php一样的位置。

text.php中修改数据库连接

数据库中有四个字段

id ,url,name,location,

url:网址链接，

name:名称

location：需要放置的位置，如填写'1'，那么位置就是第一列，（注意：目前就总共3列，也就是说你的值只能填写1,或者2，或者3）
